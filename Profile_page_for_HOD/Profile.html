Instructions

Welcome! Please read the instructions carefully before proceeding:

  • Approve events at least 5 days in advance.
  • Review all event details carefully before clicking Approve or Reject. Once the action is taken, it cannot be changed.
  • Dark Red row indicates the event cannot be approved (less than 5 days remaining).
  • Light Red row means you have very limited time left to approve.
  • White row means the event is within a safe approval window.
  • Once an event is approved or rejected, you cannot update it.
  • You can view the Participants File for Completed Events in the Profile Page.
  • If you try to delete your account and there are Pending Events, the Event Hosters will be notified via email, and Rejected Events will be permanently deleted.
  • If there are Approved but Upcoming Events, you cannot delete your account until those events are marked as Completed.
