Instructions

Welcome! Please read the instructions carefully before proceeding.

For Event Hosts:
  • Submit your event proposal at least 15 days before the scheduled event date.
  • Ensure that all event details (title, date, venue, and description) are accurate and complete.
  • Once submitted, the event will be reviewed by the HOD. You cannot edit the event details after submission.
  • Keep track of your event status (Pending, Approved, Rejected) through your dashboard.
  • You will receive a notification once your event is approved or rejected.
  • If rejected, you may resubmit a corrected version if time permits.
  • If event is Approved and Not Completed, you cannot delete the event or your account.
  • If your event is still Pending and the HOD deletes their account, your event will also be deleted and you will be notified via email.
  • Once an event is added, it cannot be updated. So verify everything before submitting.
  • You can only Add and Delete events, not Approve or Update them.
  • If you have Paid for the event, you must collect fees manually; there is no online method.
  • Students can enroll in your event before the event day; enrollment is not allowed on the event day.
  • You can Download the participants file from your Profile Page.
  • For your event, Students can generate their Certificates. A Sample is provided.
  • You can read FAQs in the Profile Page also.
For Students:
  • Each event card will show a badge indicating whether the event is Paid or Free.
  • If you enroll in a Paid Event, you must give the specified amount directly to the event host on the event day.
  • If you enroll in a Free Event or Paid Event but do not attend, your account will be blocked.
  • You can generate certificates only after the event is marked as Completed.
  • Read the event details and ensure you are eligible before enrolling.
  • Enrollment is allowed only before the event day.
